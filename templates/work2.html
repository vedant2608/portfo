002 : Website Scrapper using Python

A simple web scrapping script which will scrape the data from the the best article source HackerNews. This Script will ask you the page number and will automatically shows the best articles having votes more than 99 on the console with title of the article, what is the link and the number of votes for that article. I scraped the website according to the robots.txt.
